## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Fonts**: Inter, JetBrains Mono

## Customization

1. **Update Personal Info**: Edit the content in each component file
2. **Change Colors**: Modify Tailwind config or CSS variables
3. **Add Sections**: Create new components and import them in `index.tsx`
4. **Update Links**: Change social media and contact links in `Contact.tsx` and `Navigation.tsx`

- Minimal bundle size
